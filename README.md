# Mini-Spark: Batch DAG Engine (Proyecto SO)

Este proyecto implementa un motor distribuido estilo Spark para ejecutar DAGs por lotes (batch), incluyendo:

- Master node
- Worker nodes en Docker
- Registro de workers y heartbeats
- Scheduler básico
- Ejecución de tareas (map, flatMap, filter, reduce)
- Ejecución completa de DAGs con dependencias
